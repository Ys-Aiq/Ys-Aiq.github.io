##手机验证码接口##

###获取随机验证码###

  • 接口名称：获取随机码
  • 接口请求地址 ： http://112.74.114.123:6379/api/send
  • 请求方法：POST 、GET
  • 说明：用户输入接口相应的参数，来获取验证密码，验证码发送到目的手机，同时返回该验证码。

接口调用注意事项：

  • 同一请求号码2分钟内只能获取一次验证码，一天最多可获取5次验证码。

请求参数：

参数名 参数类型 必填 描述 默认值
phone string true 用户手机号码，11位有效数字组成的合法手机号 无

返回成功示例

参数说明：

code : 结果代码。
mes_code : 获取的随机验证码，验证码5分钟内有效。
